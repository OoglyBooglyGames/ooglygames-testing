Console Login Required

This Alpine Linux instance requires a manual login after boot.

📝 Follow these steps exactly:

1. Wait for the kernel messages to finish.

2. At the login: prompt, type:

vflogin yap

3. Press Enter.

4. When prompted for Password:, type:

123456

5. Press Enter again.

The console is case-sensitive. Type carefully.
